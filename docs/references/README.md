# 参考文档

本目录存放第三方库和工具的参考文档，主要为 LLM 友好的 `*-llms.txt` 格式。

## 为什么要在这里存放第三方文档？

1. **离线可用**：Agent 无需联网即可了解依赖库
2. **版本锁定**：文档与项目使用的库版本匹配
3. **上下文优化**：`llms.txt` 格式专为 LLM 优化，信息密度高

|------|--------|------|------|
| - | - | - | 暂无 |

## 如何添加新的参考文档

1. 查找库是否提供 `llms.txt` 文件
2. 如果有，下载并重命名为 `<库名>-llms.txt`
3. 如果没有，考虑创建一个精简版本：
   - 核心 API 用法
   - 常用配置选项
   - 本项目的使用方式
